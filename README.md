This repository will discuss the implementation and verification of a single cycle 32-bit RISC-V processor.
The entire implementation of the processor is divided into control path (control unit) and datapath (register file, data memory, ALU, ALU control unit, instruction memory, program counter and other supporting blocks).

Register block- 
Small, very fast memory present inside the CPU. 
It contains CPUs present working data. 
Any instruction can read the data from the register file or write the data into this register file.
